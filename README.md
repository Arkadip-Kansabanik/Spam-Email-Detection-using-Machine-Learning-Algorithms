# Spam-Email-Detection-using-Machine-Learning-Algorithms
This project implements an NLP-driven spam detection system using machine learning to classify email and SMS messages. It utilizes text preprocessing, TF-IDF feature extraction, and a comparative analysis of supervised models. Bernoulli Naive Bayes achieved the highest performance with 97.7% accuracy.
# What I learned from this project
I gained hands-on experience with NLP techniques like TF-IDF and text preprocessing to build a robust spam classifier. I learned to perform comparative analysis across eight machine learning models, ultimately identifying Bernoulli Naive Bayes as the most effective for high-precision, real-time message identification.
